#  Web Application Project

A web application built with:
- Django (Web Framework)
- MySQL (Database)
- Google Cloud Platform
  - App Engine (Deployment)
  - Cloud SQL (Database hosting)
- Large Language Models (LLM) Integration

## Project Overview
This project combines modern web technologies with AI capabilities to create an interactive web application.

## Technologies
- Backend: Django
- Database: MySQL
- Cloud Platform: Google Cloud (App Engine + Cloud SQL)
- AI Integration: Large Language Models
- Frontend: HTML, CSS, JavaScript



## Development
- Local development server: `python manage.py runserver`
- Deployment: `gcloud app deploy`
